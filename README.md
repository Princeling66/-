# 微信小程序_个人简历

#### 介绍
基于微信云开发写的一个个人简历微信小程序，组件库使用的是ColorUI。


#### 安装教程

1.  在微信开发者工具导入整个Resume文件，代码方法都写了注释，由于数据存到云数据库和云仓库，可能有些内容看不到，可以扫码预览一下内容。觉得对你有帮助的麻烦给个Star,谢谢！！！
